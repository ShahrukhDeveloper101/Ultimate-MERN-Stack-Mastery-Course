When you set an element's position to absolute, it positions the element relative to its nearest positioned ancestor. An ancestor element is considered positioned if its position property is set to relative, absolute, fixed, or sticky.

The default value of the flex-direction property in CSS is row . This means that items within a flex container are laid out along the main axis in the direction of the inline axis of the writing mode.
